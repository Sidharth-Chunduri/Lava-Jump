This file was created for FBLA Computer Game and Simulation porgramming 2022 using Unity game engine. This is a 1980's style arcade inspired game.

Controls: WASD to move. SPACE to Jump. For Multiplayer: Player2 uses Arrow keys to move and Right Control to jump 

Running the Game:
open the game folder and locate LavaJump.exe. Right click the file and hit open.
